# Medical Project

A full-stack medical appointment booking system with admin panel and frontend.

## Project Structure

- **admin/** - Admin dashboard for managing doctors and appointments
- **vite-project/** - Frontend application for patients
- **Backend/** - Node.js backend server
- **assets/** - Static assets and images

## Setup Instructions

### Backend
```bash
cd Backend
npm install
npm start
```

### Frontend
```bash
cd vite-project
npm install
npm run dev
```

### Admin Panel
```bash
cd admin
npm install
npm run dev
```

## Environment Variables

Create `.env` files in each directory (Backend, admin, vite-project) with required configurations.
